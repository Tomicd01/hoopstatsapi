# HoopStats API

A RESTful Web API for managing basketball league statistics, providing comprehensive functionality for tracking players, games, teams, and statistical data in basketball competitions.

## Project Overview

HoopStats API is currently in development, designed to serve as a robust backend solution for basketball statistics management. The project implements Clean Architecture principles and Repository Pattern to ensure maintainable, testable, and scalable code.

## Architecture

The solution follows Clean Architecture with distinct layers:

- **API (Host)**: Handles HTTP requests and responses
- **Application**: Contains business logic, DTOs, and service interfaces
- **Domain**: Core business entities and business rules
- **Infrastructure**: Implements data access, migrations, and external services

## Key Features

- Player management and statistics
- Game statistics tracking
- Team management
- Season tracking
- Position-based player categorization
- Comprehensive statistical analysis

## Technical Stack

- .NET 9.0
- Entity Framework Core
- AutoMapper for object mapping
- RESTful API principles
- Clean Architecture
- Repository Pattern
